# Code Book for UCI Human Activity Using Smartphones

### Variables
Chose variables that have either "mean" or "std" in their name
Elements of variable names:
* domain
  * **time**
  * **frequency**
* acceleration signal
  * **body**
  * **gravity**
*  device
   * **accelerometer**
   * **gyroscope**
* type of variable
   * **3axial[modifier]**
     * [modifier] = X, Y, or Z appended to end
   * **magnitude**
   * **jerk3axial[modifier]**
     * [modifier] = X, Y, or Z appended to end
   * **jerkmagnitude**
